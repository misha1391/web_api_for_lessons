Это проект, показывающий, как работает FastAPI и Uvicorn на примере сайта для школы.

В этом проекте есть уже:
- Учетная запись
- Управление оценками, классами, событиями, уроками
- Добавление электронной почты(Пока ни для чего не используется)

В процессе:
- Смена пароля

В будующем будет:
- Проверка электронной почты
- Разделение учетных записей для учеников и учителей
- Смена пароля, если пользователь забыл свой пароль(по email)

Как установить библотеки? - Они уже есть в папке .venv, но можно установить и вручную:
- FastAPI - ввести в cmd команду > pip install fastapi
- Uvicorn - ввести в cmd команду > pip install uvicorn

Необходимые версии:
- python 3.10+ (в .venv используется 3.13.7)
